When people hear "type" or "typography" they usually think of Microsoft Word or PowerPoint, something decorative such as this. When in reality type is something much more detailed, skilled and advanced.
Type designers are a highly skilled group inside the world of typography that makes typographic systems. They start out by creating individual letterforms and then
build up to something more complex with several characters, weights, spacing, hinting and OpenType features.

Fonts
Formatted
For the Web

Webfonts are just fonts formatted for the web and can come in many forms: OT, EOT (Embedded OpenType), WOFF and SVG. The main difference is basically compression, structure and browser support.

Sexy

When looking for a good webfont you want to find a sexy one. And by sexy I mean small in file size (compression) & loaded with extra characters. That is if you are going to use them.

Junk in
the trunk.

Extended characters such as ligatures, real fractions, stylistic sets, small caps, etc.

These extra characters are known as OT features. They enhance type legibility and give greater typographic layout controls. The features include ligatures, fractions, stylistic sets, smalls caps and many other.

The experimental property I’m going to focus on is the font feature settings. When I say experimental I mean it is in the Editor’s Draft of the W3C (World Wide Web Consortium) and not currently a recommendation.
This is where you turn to the person sitting next to you and say "Man, this guy is crazy."

Yes, you will need browser prefixes!

The code is relatively simple because it uses the existing OT tags as used by type designers when creating OT data structures. The website I created to showcase the features has a couple different sections. The first section displays the features next to its CSS code.
It also has a section where I asked leading experts questions relating to web type and basically any helpful knowledge on the subject. Finally there is a blog section where I discuss and share additional findings on the topic. There can be a lot of weird and specific things that can happen when using webfonts so this section addresses those issues.
